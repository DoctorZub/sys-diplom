### Введение
Ключевая задача данной работы — это разработать отказоустойчивую инфраструктуру для сайта, включающую мониторинг, сбор логов и резервное копирование основных данных. Инфраструктура будет размещаться в Yandex Cloud. В качестве тестового сайта будет выступать приветственная страница сервера Nginx, для мониторинга будет использоваться Zabbix Server и Zabbix Agents2, для сбора и анализа логов будет использоваться ELK стек, в качестве технологии резервного копирования выбрано создание snapshot'ов дисков всех ВМ.
